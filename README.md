# Lead Management Tool – Protótipo

Protótipo em HTML de ferramenta para gestão de leads e cold calls para transportadora.

## Objetivo

Simular fluxo de trabalho de:

- Registro de leads
- Histórico de tentativas de contato
- Priorização de follow-up

## Stack

- HTML simples (protótipo UI/UX)
- Pensado para futura migração para backend (Python/FastAPI + banco de dados)

## Status

Protótipo validado com usuário real. Empresa encerrou atividades antes da migração para sistema completo.

## Como usar

1. Baixe o Index.html
2. Abra o Index.html no navegador da sua preferência para testar o protótipo.
